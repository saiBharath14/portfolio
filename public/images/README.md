# Project Images

Add your project screenshots here with the following naming convention:

## Required Images:

1. **bookmart-fullstack.jpg** - Bookmart MERN application screenshot
2. **task-management.jpg** - Task Management System screenshot  
3. **fertile-minds-ui.jpg** - Fertile Minds UI/UX design screenshot
4. **real-estate.jpg** - Real Estate Portal screenshot
5. **lms.jpg** - Learning Management System screenshot

## Image Requirements:

- **Format:** JPG, PNG, or WebP
- **Dimensions:** 600x400 pixels (recommended)
- **File Size:** Under 500KB per image
- **Quality:** High resolution, clear screenshots

## How to Add Images:

1. Save your project screenshots with the exact names listed above
2. Place them in this `public/images/` folder
3. The images will automatically appear in your portfolio

## Fallback:

If an image is missing, the portfolio will show a fallback with technology icons.

## Example:
```
public/
  images/
    bookmart-fullstack.jpg
    task-management.jpg
    fertile-minds-ui.jpg
    real-estate.jpg
    lms.jpg
```
